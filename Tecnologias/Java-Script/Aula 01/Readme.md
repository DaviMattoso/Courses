Este repositório reúne todo o meu material de estudo sobre JavaScript, desenvolvido de forma autodidata com base em cursos, documentação oficial, vídeos, artigos e comunidades como GitHub e Stack Overflow. Todo o conteúdo está em português e organizado da forma que faz mais sentido para o meu processo de aprendizado.

# 💡 Por que estudar JavaScript?

HTML cria a estrutura.

CSS deixa tudo bonito.

Mas é o **JavaScript** que faz a mágica acontecer.

É com ele que criamos páginas interativas, formulários inteligentes, animações, consumo de APIs e aplicações completas. Como estou focando no desenvolvimento web, estudar JavaScript é um passo essencial para continuar evoluindo.

Sem falar que é uma linguagem bastante compativel com bases de dados como MySQL e outras
